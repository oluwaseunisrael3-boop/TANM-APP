## 📱 HOW TO INSTALL THE APP ON YOUR PHONE

The TANM Ministry App works like a native app on any phone — **no App Store needed.** Just follow the steps for your phone type below.

---

### 🤖 FOR ANDROID PHONES (Samsung, Tecno, Infinix, iPhone, etc.)

**Using Google Chrome (recommended):**

1. Open **Google Chrome** on your phone
2. Go to: `https://oluwaseunisrael3-boop.github.io/TANM-App/TANM-Ministry-App.html`
3. Wait for the page to fully load
4. Tap the **three dots menu** (⋮) at the top right of Chrome
5. Tap **"Add to Home screen"**
6. Tap **"Add"** when the prompt appears
7. The **TANM app icon** will now appear on your home screen
8. Tap it to open the app — it works like a real app!

---

### 🍎 FOR iPHONE / iPAD (Apple devices)

**Using Safari (must use Safari — not Chrome):**

1. Open **Safari** browser on your iPhone/iPad
2. Go to: `https://oluwaseunisrael3-boop.github.io/TANM-App/TANM-Ministry-App.html`
3. Wait for the page to fully load
4. Tap the **Share button** (the box with an arrow pointing up) at the bottom of the screen
5. Scroll down and tap **"Add to Home Screen"**
6. Type **"TANM"** as the name (or leave as is)
7. Tap **"Add"** in the top right corner
8. The **TANM app icon** appears on your home screen
9. Tap it anytime to open the ministry app!

---

### 💻 ON A COMPUTER / LAPTOP

1. Open **Google Chrome**
2. Visit: `https://oluwaseunisrael3-boop.github.io/TANM-App/TANM-Ministry-App.html`
3. Look for the **install icon** (➕) in the browser address bar
4. Click it and select **"Install"**
5. The app opens as a standalone window on your desktop

## ⚙️ ADMIN DASHBOARD GUIDE

**For Rev. Bosun Afolayan only**

1. Tap the **Admin** tab in the top navigation menu
2. Enter the admin password
3. Choose what you want to manage using the 8 tabs:

| Tab | What You Can Do |
|-----|-----------------|
| 🔥 Prophetic | Write and post daily prophetic declarations |
| 🎙 Sermons | Write full sermon text + add audio/video link |
| 📡 Teachings | Write and post daily teachings |
| 📖 Devotional | Add new devotional entries |
| 📅 Events | Add upcoming programs and events |
| 📚 Resources | Add e-books, audio, and study materials |
| 👥 Community | Delete any inappropriate community posts |
| 🎓 Courses | View all 22 course access codes |

### How to Post a Prophetic Word:
1. Tap Admin → enter password
2. Tap **Prophetic** tab
3. Select today's date
4. Enter the Scripture reference (e.g. Isaiah 60:1)
5. Write the full prophetic word in the text box
6. Tap **"Post Prophetic Word"**
7. It appears instantly on the Prophetic page and Home page

## 🚀 UPLOADING TO GITHUB

1. Go to [github.com](https://github.com) and login to your account
2. Create a new repository named **`TANM-App`**
3. Upload all these files to the repository:
   - `TANM-Ministry-App.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
   - `README.md`
4. Go to **Settings → Pages**
5. Under "Source", select **main branch** and click **Save**
6. Your app will be live at:
   `https://oluwaseunisrael3-boop.github.io/TANM-App/TANM-Ministry-App.html`

> **Upload tip:** On mobile, open each file in GitHub, tap the **pencil icon** or use the three-dots rename trick to upload files one by one.
